# Azure Python Web App

This is a simple Flask web application deployed using Azure App Service.

## Purpose

This project is created for practicing deployment and understanding how Azure App Service works with GitHub integration.

## Tech Stack

* Python
* Flask
* Azure App Service

## Deployment

The application is connected to GitHub and deployed automatically using Azure Deployment Center.

## Output

When deployed successfully, the app displays a simple message in the browser.
